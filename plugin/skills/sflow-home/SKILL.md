# Singularity Flow cockpit

Run `singularity-flow cockpit`. Reproduce the current phase, progress, persona, assignment, and next actions in the visible response. Do not mutate lifecycle state. If setup is incomplete, follow with `singularity-flow doctor` and explain only the failed or warning checks.
